# 🧠 TherapBot

**RL-Based Empathetic Mental Health Chatbot**

TherapBot is an intelligent conversational agent designed to provide empathetic mental health support using NLP and reinforcement learning concepts. It leverages DialoGPT for human-like responses and TextBlob for real-time emotion detection.

The system identifies user emotions and dynamically adapts responses, offering personalized coping strategies such as breathing exercises, grounding techniques, and positive reinforcement. A lightweight reinforcement learning feedback loop enables continuous improvement based on user feedback.

---

## 🚀 Features

* 🤖 Empathetic response generation using transformer models
* 🧠 Emotion detection (keyword + sentiment analysis)
* 💬 Context-aware conversation with memory handling
* 🔁 Reinforcement learning-based feedback system
* 🌿 Personalized coping strategies based on emotions
* ⚡ Lightweight and efficient implementation

## 💡 How It Works

1. User inputs a message
2. Emotion is detected using keywords + sentiment polarity
3. Conversation history is maintained for context
4. Model generates an empathetic response
5. Bot optionally suggests coping strategies
6. User feedback updates reinforcement learning reward
